About

  • Written entirely from scratch using C++.

Download

Below are steps to running the app. If you are an experienced git user, just use git to clone the repository. Otherwise, follow the Poor mans guide below.


Poor mans steps to running app

  1. Download and install Qt. (click icon directly below)
  2. Download and extract zip (click icon directly below).
  3. Double-click CG_mobilechess.pro (located in your_install_path\play-zone\src\CG_mobileChess\CG_mobilechess.pro).
  4. Press Ctrl + R (i.e. this builds and runs the program).
  5. Register account or just login with default user.
  6. Login.
  7. Enter game queue (1-minute, 5-minute, 30-minute).
  8. Play chess.
